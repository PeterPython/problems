This is a program (written in Java) to manage a personal library that stores titles and authors.

To run this program you first have to compile the .java file by using this command in the command line:
$javac Library.java

Then run the program with this command:
$java Library

The program will be running until the 'quit' command is used

To use a command, just type the command with the appropiate format. e.g: 
add "The Grapes of Wrath" "John Steinbeck"
show all
